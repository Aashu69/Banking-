
# Banking Transaction Monitoring System

## Overview
This project analyzes banking transactions and detects suspicious transactions
based on logical conditions (very high amounts). It uses Python, Pandas, and Matplotlib.

## Features
- Load transaction dataset
- Detect suspicious transactions
- Basic statistics
- Transaction amount visualization

## Installation

pip install -r requirements.txt

## Run

python main.py

## Suspicious Logic
Transactions greater than 15000 are marked suspicious.

## Deployment
Works locally and can be pushed to GitHub easily.
